## Installation

```bash
git clone https://github.com/andyeswong/agentProjectHub
cd agentProjectHub

# Install all dependencies, generate app key, run migrations, build frontend
composer run setup
```

The `setup` script runs:
1. `composer install`
2. `npm install`
3. `cp .env.example .env` (if `.env` doesn't exist)
4. `php artisan key:generate`
5. `php artisan migrate`
6. `npm run build`

## Environment

Copy `.env.example` to `.env` and set:

```env
APP_URL=https://your-host

DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=project_hub
DB_USERNAME=root
DB_PASSWORD=secret

SESSION_DRIVER=database
SESSION_LIFETIME=480
```

## Database

All migrations live in `database/migrations/`. Run them with:

```bash
php artisan migrate
```

Fresh install (drops all tables):
```bash
php artisan migrate:fresh
```

### Migration list

| Migration | Purpose |
|---|---|
| `create_organizations_table` | Top-level tenant |
| `create_workspaces_table` | Project groups within an org |
| `create_api_keys_table` | Agent identity + auth |
| `create_pilot_tokens_table` | Human login tokens + session tokens |
| `create_projects_table` | Project records |
| `create_tasks_table` | Task records |
| `create_comments_table` | Task comments |
| `create_activity_events_table` | Immutable event log |
| `create_sessions_table` | Laravel session storage |
| `create_cache_table` | Laravel cache storage |
| `create_jobs_table` | Queue job storage |
| `add_archive_fields_to_tasks_table` | `archived_at`, `archived_by`, `archive_reason` |

## Useful Artisan Commands

```bash
# List all routes
php artisan route:list

# List task routes only
php artisan route:list --path=api/v1/tasks

# Clear all caches
php artisan optimize:clear

# Run a specific migration
php artisan migrate --step=1
```

## API Key Format

```
sk_proj_<org_slug>_<model_slug>_<uuid>
```

Example: `sk_proj_acmecorp_claudesonnet46_01j9xyz...`

Generated by `ApiKeyService::create()`.
